GetData_CourseProject_set2014

This repository contains the information, script, data set and code book related to the “Getting and Cleaning Data” course project which was developed according to the professor’s instructions.

The data used for this assignment were originated from a project aimed at studying the Human Activity Recognition Using Smartphones developed by Reyes-Ortiz et al 2012. This data set was collected, cleaned, edited and merged to produce a new tidy data, useful for further analysis.

The Code book contains more details regarding the original experiment, variables and the procedure used in this work. The file run.analysis.R contains the script, and the file tidy.csv the tidy data set obtained.
